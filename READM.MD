# Custom Button Designs

## Description

A simple frontend project that demonstrates the creation and styling of custom buttons using HTML5 and CSS3.

The project includes different button styles such as search, plus, and create account buttons with customized layouts and visual effects.

## Technologies Used

- HTML5
- CSS3

## Features

- Custom styled buttons
- Clean and simple UI design
- CSS-based layouts and effects
- Beginner-friendly frontend component practice
